# Simple Interest Calculator

A simple command-line application or script written to calculate simple interest based on the principal amount, annual interest rate, and time period in years.

## Formula
The formula used to calculate simple interest is:
I = P * R * T / 100

Where:
- **I** = Interest amount
- **P** = Principal amount (initial investment or loan amount)
- **R** = Annual interest rate (in percentage)
- **T** = Time period (in years)

## Input Parameters
- `principal`: The initial amount of money.
- `rate`: The interest rate per year.
- `time`: The duration in years.

## Output
- Returns the total interest earned/owed over the specified period.
